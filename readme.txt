Este proyecto consiste en el desarrollo de un sitio web completo de e-commerce que combine diferentes tecnologías web, mi caso sera una tienda de ropa virtual llamada "Selfless". El objetivo es crear una plataforma dinámica e interactiva que consuma datos de una API REST para mostrar productos, y que permita a los usuarios añadir productos a un carrito de compras. El diseño de la página será completamente responsivo, garantizando que se adapte correctamente a diferentes dispositivos y tamaños de pantalla. Además, se priorizará el uso de una estructura HTML semántica para asegurar la interpretación del contenido, tanto por los usuarios como por los motores de búsqueda, mejorando así la accesibilidad y el SEO del sitio.
En cuanto a las funcionalidades principales, los usuarios podrán navegar por un catálogo de productos. También se implementará un carrito de compras funcional que permitirá agregar, eliminar y gestionar productos seleccionados antes de realizar una posible compra. El proyecto incluirá el uso de CSS avanzado para lograr un diseño moderno y atractivo, y JavaScript para la manipulación del DOM y la creación de interacciones dinámicas.
